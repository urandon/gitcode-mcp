# Project Management

Lightweight repo-local project management for `gitcode-mcp`.

- `tasks/`: implementation backlog and task files.
- `research/`: API discovery, experiments, and notes.
- `decisions/`: small ADR-style decisions.
- `handoffs/`: session handoffs and acceptance notes for this tooling repo.

Keep this repo focused on reusable tooling implementation. Project-specific decisions, acceptance truth, and cutover planning belong outside this public tooling repository.
